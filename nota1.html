Nota: Membina Pautan Teks dan Imej dalam Atur Cara HTML

1. Konsep Pautan (Hyperlink)

Pautan (Hyperlink): Elemen dalam laman web yang menghubungkan ke halaman lain, dokumen, imej, atau sumber luar.

Dua Jenis Pautan:

  • Pautan Teks: Pautan menggunakan teks sebagai elemen klik.
  • Pautan Imej: Pautan menggunakan imej sebagai elemen klik.

2. Membina Pautan Teks

<a href="URL">Teks Pautan</a>

Contoh:

<a href="https://www.google.com">Lawati Google</a>

3. Membina Pautan Imej

<a href="URL">
  <img src="URL_IMEJ" alt="Teks Alternatif">
</a>

Contoh:

<a href="https://www.wikipedia.org">
  <img src="logo.png" alt="Logo Wikipedia">
</a>

4. Pautan Relatif dan Mutlak

Pautan Relatif:

<a href="halaman2.html">Halaman 2</a>

Pautan Mutlak:

<a href="https://www.example.com">Halaman Utama</a>

5. Membuka Pautan dalam Tab Baru

<a href="https://www.youtube.com" target="_blank">YouTube</a>

6. Gaya Pautan

Pautan boleh digayakan dengan CSS:

a {
  color: blue;
  text-decoration: none;
}
a:hover {
  color: red;
  text-decoration: underline;
}

7. Kepentingan dan Aplikasi

Memudahkan navigasi dalam laman web dan membantu pengguna mengakses maklumat tambahan atau sumber luar.
